HEATING & COOLING

4. Fan speed: Auto (recommended) or Low/Medium/High

Heating Mode:
1. Press "Heat" button on remote
2. Set temperature (recommended: 68°F)
3. Press "On" button

Central Heating:
- Thermostat location: Hallway, near bathroom
- Model: Carrier Infinity
- Set desired temperature using up/down arrows
- System automatically maintains temperature

Troubleshooting:
- If AC makes strange noises: May need cleaning, contact landlord
- If AC doesn't cool: Check filter (behind front panel), clean if dirty
- If heating doesn't work: Check thermostat batteries, ensure system is ON
- If unit is very loud: Normal for first few minutes, should quiet down
- If AC leaks water: Check drain pan, contact landlord if issue persists

LAUNDRY
-------
Washer: Front-loading, located in hallway closet
Dryer: Electric, located next to washer

How to Use Washer:
1. Load clothes into washer
2. Add detergent to dispenser drawer (top left)
3. Select cycle: Normal (default), Delicates, Heavy Duty, Quick Wash
4. Select water temperature: Cold (recommended), Warm, Hot
5. Press "Start" button
6. Cycle takes 30-60 minutes

How to Use Dryer:
1. Load clothes from washer
2. Clean lint trap (pull out filter, remove lint, replace)
3. Select heat setting: High (default), Medium, Low, Air Dry
4. Select time: Auto (recommended) or manual time
5. Press "Start" button
6. Check clothes periodically - remove when dry

Important:
- Always clean lint trap before each use
- Don't overload washer or dryer
- Use appropriate detergent (HE detergent for front-loading washer)
